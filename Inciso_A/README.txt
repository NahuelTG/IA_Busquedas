Pasos para ejecutar el programa:
1) Abre la consola en la ubicación donde se encuentran los archivos.py (Inciso_A)
2) Instala las siguientes bibliotecas ejecutando los siguientes comandos:

    Para instalar NetworkX:

        pip install networkx

    Para instalar Matplotlib:

        pip install matplotlib

3) Ejecuta el siguiente comando para iniciar el programa:

    python inicio.py

4) Luego siga las instrucciones para que el codigo funione.

5) Disfruta :)
