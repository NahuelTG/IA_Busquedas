pasos para correr el programa:
1) abrir la consola donde estan los archivos.py
2) instalar las siguientes bibliotecas
    pip install copy
    pit install anytree
3) en caso de estar instaladas actualizarlas
    pip --upgrade copy
    pip --upgrade anytree
4) colocar el siguiente comando para arrancar
    python programa.py
5) llenar los datos que se piden por ejemplo:
    Ingresa el tamanio de la habitacion (en metros): 3
    Ingresa en que posicion (en metros) estara el mono: 2
    Ingresa 0 para que este en el suelo o 1 para que este un nivel mas, caja1: 0
    Ingresa en que posicion (en metros) estara la primera caja: 0
    Ingresa 0 para que este en el suelo o 1 para que este un nivel mas, caja2: 1
    Ingresa en que posicion (en metros) estara la segunda caja: 0
    Ingresa en que parte del techo (en metros) estara los platanos: 0   

6) La matriz se mostrara y los objetos se representaran de la siguiente forma:
    mono sera el nro 1
    caja1 sera el nro 2
    caja2 sera el nro 3
    platanos sera el nro 4

7) Disfruta :)
