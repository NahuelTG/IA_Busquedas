Pasos para ejecutar el programa del MONO EN LA HABITACION:
1) Abre la consola en la ubicación donde se encuentran los archivos.py (Inciso_B)
2) Instala las siguientes bibliotecas ejecutando los siguientes comandos:

    pip install copy
    pip install anytree

3) En caso de que las bibliotecas ya estén instaladas, puedes actualizarlas a la última versión con los siguientes comandos:

    pip --upgrade copy
    pip --upgrade anytree

4) Ejecuta el siguiente comando para iniciar el programa:

    python programa.py

5) Llena los datos que se solicitan, por ejemplo:

    Ingresa el tamanio de la habitación (en metros): 3
    Ingresa en que posición (en metros) estara el mono: 2
    Ingresa 0 para que este en el suelo o 1 para que este un nivel más, Caja_1: 0
    Ingresa en que posición (en metros) estara la primera caja: 0
    Ingresa 0 para que este en el suelo o 1 para que este un nivel más, Caja2_: 1
    Ingresa en que posicion (en metros) estara la segunda caja: 0
    Ingresa en que parte del techo (en metros) estará los platanos: 0   

6) La matriz se mostrara y los objetos se representaran de la siguiente forma:
    mono sera el nro 1
    caja1 sera el nro 2
    caja2 sera el nro 3
    platanos sera el nro 4

7) Disfruta :)
